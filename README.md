# Sales-Data-Analysis-in-Power-BI
Brazilian E-Commerce Sales Analysis

## Description
This Project is based on the Brazilian E-Commerce Public Dataset (Olist)
The goal is to analyse sales ata and generate insights about customer behavior, orders, and business performance.

## Features
- Sales performance analysis
- Customer segmentation
- Order trends over time
- Revenue and KPI tracking
- Geographic analysis of customers

## Tools & Technologies
- Power BI
- Data cleaning & transformation
- Data modeling
- DAX for calculations

## Dashboard
The interactive dashboard provides insights into:
- Total sales an revenue
- Top selling products and categories
- Customer distribution
- Delivery performance
